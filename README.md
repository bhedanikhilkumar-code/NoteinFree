# Notein Free 📝

**Notein Free** is a fast, private, and fully offline mobile notepad application built with Flutter. It prioritizes user privacy by storing all data locally on the device without requiring any accounts or cloud synchronization.

## 🚀 Features

- **Text Notes**: Create, edit, and organize text notes easily.
- **Checklists**: Manage your tasks with interactive to-do lists.
- **Calendar Reminders**: Set up date-specific reminders so you never miss an important event or task.
- **100% Offline & Private**: No cloud sync, no tracking. All your data is securely saved locally on your device using `shared_preferences`.
- **Fast & Lightweight**: Designed for high performance and low resource usage.
- **Clean UI**: A beautifully crafted, modern, and intuitive user interface for a premium experience.

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev/)
- **Language**: Dart
- **Storage**: `shared_preferences` (Local Key-Value Store)

## 📥 Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your machine.
- Android Studio or VS Code with Flutter extensions.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bhedanikhilkumar-code/NoteinFree.git
   ```
2. Navigate to the project directory:
   ```bash
   cd NoteinFree
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
